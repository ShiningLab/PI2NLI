# PI2TI
This repository is for the paper Paraphrase Identification via Textual Inference.

## Dependencies
+ python >= 3.10.10
+ torch
+ wandb
+ lightning
+ transformers
+ sentencepiece

## Setups
```
$ pip install --upgrade pip
$ pip install protobuf==3.20.*
```
